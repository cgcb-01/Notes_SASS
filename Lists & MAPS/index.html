HERE WE WILL EXPLORE LISTS & MAPS IN SASS

In sass we can have lists and we can use the values in the lists in our styles. there are many built in functions of lists.

1. length() 2. nth() 3. set-nth() 4.join() 5. append() etcs

In MAPS we can define various different types of data types in a particular lists only. The built in functions are:
1. map-get() 2.map-keys() 3.map-values() 4.map-has-keys().
